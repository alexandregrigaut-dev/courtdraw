Features

Everything a Coach Needs. Nothing They Don't.

Purpose-built for coaching, not graphic design. Fast, focused, and frustration-free.

🏟️

37+ Regulation Courts

Every court drawn to correct proportions — full pitch, half-court, or zoomed penalty area. Switch sport in one tap.

✏️

8 Drawing Tools

Solid arrows for runs, dashed arrows for off-ball movement, arcs, lines, circles, rectangles, and freehand text labels.

🧑‍🤝‍🧑

Draggable Player Tokens

8 blue and 8 red tokens by default — drag to any position. Labels auto-fit. Works with touch and mouse equally well.

📚

Tactic Library

Save any board state — court, players, drawings — as a named tactic. Load it back in one click. Stored locally, no cloud needed.

🖼️

PNG Export

Export a high-res 1200×720 PNG of your tactic. Share directly in WhatsApp, Telegram, or paste into a presentation.

↩️

Unlimited Undo / Redo

50-step undo history. Fix a misplaced arrow or undo an accidental clear without losing your whole board.

📶

Works Offline

CourtDraw is a Progressive Web App. Install it to your home screen and draw plays pitch-side with zero signal required.

🌙

Dark & Light Mode

Dark interface by default for pitch-side use. Flip to light mode for presentations on a projector or printed handouts.

📱

Touch-First on Mobile

Built with the Pointer Events API for buttery-smooth drawing on iPad and Android tablets. No lag, no missed strokes.
